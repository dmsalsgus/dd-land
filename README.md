- 프로젝트 이름 : DD-Land 🎠
- 프로젝트 지속기간 : 2023.12.18 ~ 2023.12.27
- 개발 언어 : Java, Oracle, HTML, CSS, JavaScript, JSP
- 멤버 : 차수민, 차민재, 이정은, 박나래, 이승원, 황주원, 김형우
